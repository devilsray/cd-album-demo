Erste Umsetzung der CD-Album Aufgabe. Der Fokus wurde auf die Grundfunktionalität gesetzt:

  • Einrichtung einer MySQL Datenbank
  • Erstellung Datenstruktur mit definierten Abhängigkeiten
  • Erstellung eines Spring Boot Services (Java 8)
  • Anbindung der Datenbank mit Hibernate
  • Veröffentlichen der Datenstruktur über Rest
  • Nutzen der Datenstruktur per Ajax Frontend (CRUD)

Auf folgende Funktionalität wurde bewusst verzichtet

  • intuitive Bedienung
  • Sicherheit
  • Statusmeldungen
  • Bilder für Alben
  • Exakte und einheitliche Formatierung
  • Optische Kompatibilität für Mobilgeräte
  • Durchgehende Trennung von html / Javascript
  • DRY Prinzip wurde nicht berücksichtigt
  • Javascript wurde einfach eingesetzt, Frameworks wie AngularJs fanden keine Anwendung
